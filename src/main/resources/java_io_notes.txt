ðŸ”¹ 4. CHARACTER STREAMS

Character streams are based on:

Reader (abstract class for reading)

Writer (abstract class for writing)

They handle 16-bit Unicode characters (supporting all languages).

ðŸ§© Reader (Abstract Class)

Purpose: To read characters (not bytes).

ðŸ”¸ Common Methods:
| Method                                    | Description                          |
| ----------------------------------------- | ------------------------------------ |
| `int read()`                              | Reads a single character             |
| `int read(char[] cbuf)`                   | Reads multiple characters            |
| `int read(char[] cbuf, int off, int len)` | Reads part of array                  |
| `boolean ready()`                         | Checks if stream is ready to be read |
| `void close()`                            | Closes the stream                    |
| `long skip(long n)`                       | Skips characters                     |


ðŸ§© Writer (Abstract Class)

Purpose: To write characters.

ðŸ”¸ Common Methods:
| Method                    | Description          |

| `void flush()`            | Flushes data         |
| `void close()`            | Closes writer        |

ðŸ”¹ 5. When to Use What
| Scenario                                    | Use                                           | Why                         |
| ------------------------------------------- | --------------------------------------------- | --------------------------- |
| Reading/Writing binary (images, PDF, audio) | **Byte Streams (InputStream / OutputStream)** | Works with raw bytes        |
| Reading/Writing text (UTF-8, Unicode)       | **Character Streams (Reader / Writer)**       | Converts bytes â†’ characters |
| Faster performance with large data          | **Buffered Streams**                          | Reduces disk access         |
| Primitive or Object data                    | **Data/Object Streams**                       | For serialization           |
| Communication between threads               | **Piped Streams**                             | Enables thread I/O pipeline |
| Printing formatted text                     | **PrintStream / PrintWriter**                 | Simple and user-friendly    |
