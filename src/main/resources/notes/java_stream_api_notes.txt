| Method                                | Use                                        | Notes                               |
| ------------------------------------- | ------------------------------------------ | ----------------------------------- |
| `collection.stream()`                 | Convert any collection to stream           | Most common                         |
| `Arrays.stream()`                     | Works for arrays (Object[] or primitives)  | For `int[]`, returns `IntStream`    |
| `Stream.of()`                         | Creates a finite stream from fixed values  | Handy for testing                   |
| `Stream.builder()`                    | Build stream dynamically                   | Good for dynamic inputs             |
| `Stream.generate(Supplier)`           | Infinite stream (must limit)               | Use for random / constant sequences |
| `Stream.iterate(seed, UnaryOperator)` | Infinite arithmetic / custom logic stream  | Good for number sequences           |
| `Stream.concat(s1, s2)`               | Combine two streams                        | Works for same type streams         |
| `Stream.ofNullable()`                 | Stream of one or zero elements (null-safe) | Introduced in Java 9                |
| `IntStream.range(a,b)`                | For numeric range                          | Includes a, excludes b              |



| Concept                      | Real-Time Use                                                                       | Interview Tip                                         |
| ---------------------------- | ----------------------------------------------------------------------------------- | ----------------------------------------------------- |
| **filter()**                 | Filtering lists (active users, valid data, etc.)                                    | Often asked with `Predicate` functional interface     |
| **map()**                    | Transforming (convert entity → DTO, string → uppercase)                             | Commonly combined with `collect()`                    |
| **flatMap()**                | Flatten nested collections (e.g., Employee → List<Address>)                         | Always explain 1-to-many mapping                      |
| **distinct()**               | Remove duplicates                                                                   | Based on `equals()` & `hashCode()`                    |
| **sorted()**                 | Sorting list data                                                                   | Default uses natural order                            |
| **peek()**                   | Debug pipelines                                                                     | Do NOT use for modifying data                         |
| **limit(), skip()**          | Pagination-like operations                                                          | Common in APIs (e.g., `skip(offset).limit(pageSize)`) |
| **takeWhile(), dropWhile()** | Stream slicing (since Java 9)                                                       | Mention “short-circuiting” behavior                   |
| **Lazy Evaluation**          | Streams don’t run until a terminal operation (`forEach`, `collect`, etc.) is called | Interviewers love this question                       |
